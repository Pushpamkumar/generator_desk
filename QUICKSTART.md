# Quick Start Guide - AI Investment Desk Note Generator

## 🚀 Start in 5 Minutes

### What You Need
- Python 3.8+ installed
- Anthropic API key (get free at https://console.anthropic.com)
- ~500MB disk space

---

## Installation (Windows/Mac/Linux)

### 1️⃣ Install Dependencies (1 minute)
```bash
# Navigate to project folder
cd AI-Investment-Desk-Generator

# Create virtual environment
python -m venv venv

# Activate it
# Windows: venv\Scripts\activate
# Mac/Linux: source venv/bin/activate

# Install packages
pip install -r requirements.txt
```

### 2️⃣ Get API Key (FREE - 2 minutes)
1. Go to https://ai.google.dev/
2. Sign in with Google account
3. Click "Get API Key"
4. Copy it (keep it private!)

### 3️⃣ Run App (1 minute)
```bash
streamlit run app.py
```

Browser should open to `http://localhost:8501`

### 4️⃣ Generate First Report (1 minute)
1. Paste API key in sidebar
2. Select "Quick Sample Analysis"
3. Choose company (e.g., Tata Motors)
4. Click "Generate Desk Note"
5. Wait 30-60 seconds
6. Download as PDF/TXT/CSV

## 🔧 If Something Goes Wrong

**"Module not found"**
```bash
pip install -r requirements.txt
```

**"API Key error"**
- Verify key from console.anthropic.com
- Paste in sidebar
- Make sure it's current/not revoked

**"Port 8501 in use"**
```bash
streamlit run app.py --server.port 8502
```
